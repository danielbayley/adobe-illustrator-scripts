[Adobe Illustrator] scripts
===========================
Handy scripts for [Adobe Illustrator] on macOS.

| Menu   | Script        | Description                                       |
|:-------|:--------------|:--------------------------------------------------|
| Export | Layers to PDF | Export all _printable_ layers to a multipage PDF. |

Install
-------
`npm install --global danielbayley/adobe-illustrator-scripts`
